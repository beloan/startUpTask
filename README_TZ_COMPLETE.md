### ✅ **Пункт 5: OTP Безопасность + Метрики в товар**

#### Часть A: OTP Безопасность
- TTL кода: 5 минут (300 секунд)
- Проверка истечения: `if (Date.now() > expiresAt) → "SMS код истёк"`
- Таймер повторной отправки: 60 секунд (блокировка кнопки)
- Проверка совпадения номера и кода (phone === smsPendingPhone)
- Очистка кода из sessionStorage после верификации

**Проверка:**
```javascript
// DevTools Console:
JSON.parse(sessionStorage.getItem('sms_verification'));
// Результат: { code: "XXXXXX", phone: "79377799906", expiresAt: TIMESTAMP }

// Проверить TTL:
const sms = JSON.parse(sessionStorage.getItem('sms_verification'));
const ttl = (sms.expiresAt - Date.now()) / 1000;
console.log(`Код действует ещё ${ttl} секунд`);
```

#### Часть B: Метрики в товар
На карточке товара отображаются:
- **"Всего заказали"**: `5 прод.` (из поля `total_sold` или `sales_count`)
- **"Всего просмотров"**: `1000 просмотров` (из `view_count`)
- **"В наличии"**: `в наличии 50` (из `current_amount`)
- **"Цена выросла/упала"**: ⬆️ 15% (красный) или ⬇️ -10% (зеленый)
- **"Рейтинг"**: ⭐ 4.5 (из `avg_rating`)

**Файл:** `src/entities/product/ui/product-card.tsx` (строки 150-320)
